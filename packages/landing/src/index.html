三種備份，從輕到重

挑一種你今天有時間做的——晚點還可以接著做下一階段。

A · 5 分鐘

備份成 ZIP

輸入 username，瀏覽器抓下所有公開文章 + 圖片，打包成一個 ZIP 檔。 零帳號、零上傳。

username-matters-lifeboat.zip
含 posts/ · assets/ · MANIFEST.json · README.md · AGENT.md
開始備份 →
B · 10 分鐘

Pin 到你的 IPFS

把官方記錄的 IPFS hash 一次 pin 到你自己的 Pinata 帳號。 token 只在你的瀏覽器，關 tab 就清空。

需要：Pinata 免費帳號（1GB 永久免費，工具內附申請步驟）
設定 + dry-run →
C · 30 分鐘

蓋一個自己的站

用 ZIP 重新打包成 Astro 站台模板，丟到 Cloudflare Pages 就有 yourname.pages.dev。可以接自己的網域。

username-site.zip → drag & drop 到 CF Pages
下載站台模板 →

🤖 用 Claude / ChatGPT / Cursor 自動跑也可以—— 看 AI agent 操作手冊

📜 著作權與授權

Matters 上的文章預設為 CC BY-NC-ND 4.0 （姓名標示-非商業-禁衍生），個別作者也可能採用 CC0 或 All Rights Reserved。 每篇的實際授權都會記錄在備份檔的 frontmatter.license 與 MANIFEST.json 裡。

  • 個人收藏、本機閱讀：完全沒問題，這就是備份的本意。
  • 對外重新發布原文：請保留作者署名與原始來源連結，且不得商用。
  • 改寫、續寫、翻譯、餵 AI 訓練 / 微調 / RAG：CC BY-NC-ND 與 arr 授權的文章禁止；CC0 不受限。請逐篇確認。

詳細條款請見 Creative Commons 授權說明。 備份成功之後，README.md 與 AGENT.md 內也會附上完整指引。

透明 · 可驗證 · 你說了算

  • 完全開源：mashbean/matters-lifeboat（待轉移到 thematters/）
  • 完全 in-browser：所有抓取、打包、加密在你的 tab 裡完成
  • 零儲存：我們沒有 server，也沒有資料庫
  • token 零留存：Pinata / GitHub token 只在 sessionStorage，關 tab 就清空
  • AI-friendly：每個 flow 都有 AGENT.md，agent 能讀能跑

所以——備份你的文章，就現在。
